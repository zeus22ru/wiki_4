# Инструкция по запуску WikiQA на любом ПК под Windows

## Сборка исполняемого файла

### Шаг 1: Установка зависимостей
Откройте терминал в папке проекта и выполните:
```bash
pip install -r requirements.txt
```

### Шаг 2: Создание векторной базы данных (если ещё не создана)
```bash
python create_vector_db.py
```

### Шаг 3: Сборка исполняемого файла
Запустите скрипт сборки:
```bash
build.bat
```

Или вручную:
```bash
pyinstaller --clean wiki_qa.spec
```

После успешной сборки исполняемый файл будет находиться в папке `dist\WikiQA.exe`

---

## Запуск на другом ПК

### Требования к целевому компьютеру

1. **Windows 10 или новее** (x64)
2. **Ollama** должен быть установлен и запущен
   - Скачать: https://ollama.ai/download
   - После установки запустить Ollama
   - Установить модели:
     ```bash
     ollama pull bge-m3
     ollama pull qwen2.5:7b
     ```

### Перенос приложения

1. Скопируйте папку `dist\WikiQA` с собранным приложением на целевой компьютер
2. Убедитесь, что папка содержит:
   - `WikiQA.exe` - исполняемый файл
   - `chroma_db` - векторная база данных
   - `templates` - HTML шаблоны
   - `static` - статические файлы (CSS, JS)

### Запуск приложения

1. Запустите `WikiQA.exe` двойным кликом
2. Приложение запустится в консольном окне
3. Откройте браузер и перейдите по адресу: `http://localhost:5000`

## Возможные проблемы и решения

### Ошибка: "Не удалось подключиться к Ollama"
- Убедитесь, что Ollama установлен и запущен
- Проверьте, что Ollama доступен по адресу `http://localhost:11434`
- Попробуйте перезапустить Ollama

### Ошибка: "Векторная база данных не найдена"
- Убедитесь, что папка `chroma_db` находится рядом с `WikiQA.exe`
- Если базы нет, запустите `create_vector_db.py` на компьютере разработчика

### Антивирус блокирует запуск
- Добавьте `WikiQA.exe` в исключения антивируса
- PyInstaller иногда помечается как ложное срабатывание

### Приложение не запускается
- Убедитесь, что используется Windows x64
- Проверьте наличие прав администратора (может потребоваться для некоторых систем)

## Конфигурация

Приложение использует следующие настройки по умолчанию:

- **URL Ollama**: `http://localhost:11434`
- **Модель эмбеддингов**: `bge-m3`
- **Модель чата**: `qwen2.5:7b`
- **Порт веб-сервера**: `5000`

Для изменения настроек создайте файл `.env` рядом с `WikiQA.exe`:

```env
OLLAMA_URL=http://localhost:11434
OLLAMA_MODEL=bge-m3
OLLAMA_CHAT_MODEL=qwen2.5:7b
CHROMA_PERSIST_DIR=./chroma_db
TOP_K_RESULTS=3
```
